# Modelado con Regresión Lineal

## Objetivo
Desarrollar un modelo de regresión lineal para predecir una variable objetivo a partir de variables independientes.

## Descripción
Se implementa un modelo supervisado de regresión lineal, analizando el comportamiento de las variables y evaluando el desempeño del modelo mediante métricas de error.

## Metodología
- Preprocesamiento de datos
- Análisis exploratorio
- Entrenamiento del modelo
- Evaluación de desempeño
- Interpretación de resultados

## Archivos incluidos
- Notebook en formato .ipynb
- Reporte en formato .html
- Base de datos utilizada
