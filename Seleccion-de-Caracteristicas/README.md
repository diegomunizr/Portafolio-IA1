# Selección de Características y Resolución de Problemas

## Objetivo
Identificar las variables más relevantes para mejorar el desempeño de un modelo predictivo.

## Descripción
Se aplican técnicas de análisis y selección de características para optimizar el modelo y reducir la complejidad del problema.

## Metodología
- Análisis de correlación
- Evaluación de importancia de variables
- Comparación de modelos
- Validación de resultados

## Archivos incluidos
- Notebook en formato .ipynb
- Reporte en formato .html
- Base de datos utilizada
