# Fundamentos de Aprendizaje Estadístico

## Objetivo
Explorar los principios básicos del aprendizaje estadístico automático y su aplicación en el análisis de datos.

## Descripción
Se realiza un análisis exploratorio de datos, identificando patrones relevantes y comprendiendo la estructura del conjunto de datos utilizado.

## Metodología
- Análisis exploratorio de datos
- Visualización de variables
- Interpretación estadística

## Archivos incluidos
- Notebook en formato .ipynb
- Reporte en formato .html
- Base de datos utilizada
