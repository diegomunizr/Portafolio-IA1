# Evaluación Práctica – Modelado y Análisis de Datos

## Objetivo
Aplicar los conocimientos adquiridos en aprendizaje automático y análisis de datos para resolver un conjunto de problemas prácticos.

## Descripción
En esta evaluación se desarrollan ejercicios enfocados en:

- Análisis exploratorio de datos
- Preparación y limpieza de datos
- Aplicación de modelos de aprendizaje supervisado
- Interpretación de resultados

El objetivo principal es demostrar comprensión conceptual y capacidad de implementación práctica.

## Metodología
- Exploración de datos
- Implementación de modelos
- Evaluación de métricas
- Análisis e interpretación de resultados

## Archivos incluidos
- Notebook en formato `.ipynb`
- Reporte en formato `.html`
- Archivos complementarios (si aplica)
